
Este proyecto consiste en una tarjeta de un producto donde el usuario puede aumentar o disminuir la cantidad mediante botones. El total a pagar se actualiza automáticamente según la cantidad seleccionada.

## 🚀 Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## ✨ Funcionalidades

- Mostrar el precio base del producto.
- Aumentar la cantidad de productos.
- Disminuir la cantidad.
- Calcular automáticamente el total a pagar.
- Manipulación del DOM utilizando `querySelector()` e `innerHTML`.
